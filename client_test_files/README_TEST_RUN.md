# Тестовый запуск Realtify

Эта папка содержит файлы для первого ручного теста через `Realtify.exe`.

## Что выбрать в программе

- `PDF витяг/техпаспорт`: `01_input_pdf_apt353.pdf`
- `Номер квартири/об'єкта`: `353`
- `Тип об'єкта`: `apartment`
- `Excel шаблон`: `02_excel_template_apartment.xls`
- `Word шаблон`: `03_word_template_valuation_report.docx`
- `Папка результату`: папка `output`

После запуска программа должна создать в `output` готовый Word, Excel, скриншоты, `validation.md` и служебные JSON/MD-файлы.
